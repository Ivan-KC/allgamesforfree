# 🎮 All Games For Free

Es una aplicación frontend desarrollada con React + TypeScript + Vite que muestra juegos gratuitos y promociones activas obtenidas de APIs públicas.

El objetivo del proyecto es centralizar:

* 🆓 Juegos free-to-play
* 🎁 Giveaways activos
* 🎮 DLCs / recompensas
* 🧪 Betas abiertas

---

## 🚀 Tecnologías utilizadas

* React
* TypeScript
* Vite
* CSS modular (custom)
* APIs públicas:

  * FreeToGame API
  * GamerPower API

---

## ⚙️ Instalación y ejecución

### 1. Clonar el repositorio

```bash
git clone <URL_DEL_REPO>
cd allgamesforfree
```

---

### 2. Instalar dependencias

```bash
npm install
```

---

### 3. Ejecutar en desarrollo

```bash
npm run dev
```

---

### 4. Abrir en el navegador

```
http://localhost:5173/
```

---

## 🛠 Scripts disponibles

```bash
npm run dev     # servidor de desarrollo
npm run build   # build de producción
npm run preview # previsualizar build
```

---

## 🌐 APIs utilizadas

### FreeToGame

* https://www.freetogame.com/api/games

### GamerPower

* https://www.gamerpower.com/api/giveaways

⚠️ Nota: Se utiliza proxy en Vite para evitar problemas de CORS.
